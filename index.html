What is HTML?

HTML is a markup language that is used to create web pages and web applications.

HTML stands for HyperText Markup Language.

HyperText means that you can create links that connect to other web pages.

Markup means that you can use tags to define the structure and content of your web page.


History of HTML

HTML was invented by Tim Berners-Lee in 1991.

It has evolved through many versions since then.

The latest version is HTML5, which was published as a W3C Recommendation in 2014.
