Creating HTML file

Using Notepad in Windows

Using Run: Press Windows+R to open the Run box,type "notepad" and press Enter.
write basic structure of html save with .html extention



Using TextEdit in Mac

Using Finder: Open Finder, go to the Applications folder, and double-click on TextEdit.
configure the settings to save file as html format.
write basic structure of html and save it as file.
